dưới đây:
1. Giải thích các câu lệnh: git init: Initializes a new git repository in the current folder
 git add <file_name> : adds a specific file to the staging area 
 git add ., adds all new, modifieled, and deleted files in the current directory to the staging area
 git commit -m “message”.: creates a commit and saves the staged changes into the git repository
